No 1 About Project

The best practices in this article assume that you have:

  • Node.js and npm installed.
  • an existing Node.js app.
  • a free Heroku account.
  • the Heroku CLI. (in this case on windows, but is disponible for linux and mac)
  • an existing package.json file. The package.json file defines the dependencies that should be installed with your application.

No 2 The Process

First create a new project on Heroku website, login into Heroku CLI, after you commit your changes to git, you can deploy your app on Heroku Server.
